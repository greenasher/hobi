Hobi

Overview

Hobi Budget App is a cloud-based personal finance and budgeting application designed to help users track expenses, monitor savings, and manage budgets across multiple devices.

The application uses Firebase Authentication for secure Google Sign-In and Firestore for real-time cloud data storage and synchronization.

Features

Google Account Login
Expense Tracking
Custom Expense Categories
Budget Management
Savings Tracking
Real-Time Cloud Synchronization
Mobile and Desktop Responsive Design
Firebase Firestore Database Integration
Secure User Authentication

Technologies Used

HTML
CSS
JavaScript
Firebase Authentication
Firebase Firestore
Netlify Hosting

Project Objective

This project was developed to create a practical budgeting solution that allows users to track spending habits, manage personal finances, and access their financial information seamlessly across devices.
